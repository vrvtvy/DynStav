# Changelog

本项目所有重要变更都会记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/)，

## [Unreleased]

### Added
- **AI 模型预设供应商**：
  - 新增 13 家预设 AI 服务商（DeepSeek、Moonshot、通义千问/百炼、智谱AI、火山引擎/豆包、MiniMax、小米 MiMo、硅基流动、OpenAI、Anthropic、Google Gemini、Grok(xAI)、OpenRouter），一键添加即可使用。
  - 预设供应商无需预填模型列表，填入 API Key 后点击「获取模型列表」自动从服务商接口拉取最新可用模型，避免模型更新滞后。
  - 新增模型默认携带 `setCacheKey: true` 自定义参数。
  - 聊天模型选择器仅展示已配置 API Key 的可用供应商，底部新增「配置模型」快捷入口。
- **AI 对话多模型架构重构**：
  - 每个供应商支持配置多个模型，可独立设置 API 名称、显示名。
  - 模型级自定义请求参数（键值对），自动解析为数字/布尔/JSON 类型合并到 API 请求体。
  - 聊天输入区新增模型选择器（彩色 logo + 下拉面板），按供应商分组展示，点击切换当前模型。
  - 每个模型独立测试连接，结果实时显示在配置弹窗中。
  - 配置弹窗加宽至 70% 窗口宽度，模型编辑区支持展开/折叠，添加/展开模型时自动滚动。
  - 旧版配置自动迁移，向后兼容。
- **AI 板块分析面板**：右侧新增 AI 对话面板，基于当前板块的真实行情数据进行智能分析。
  - 支持 Chat Completions / Responses / Anthropic Messages / 自定义兼容接口四种模型模板。
  - 流式输出、一键停止、快捷提问、复制回复、重新分析等交互。
  - AI 回复附带免责声明，明确不构成投资建议。
  - 板块上下文包含每日行情数据明细表，提升分析质量。
- **字体大小调节**：三档字体大小（小 / 中 / 大），菜单栏一键切换，持久化到配置文件。
- **右侧面板收起/展开**：AI 面板可折叠收起，宽度可拖拽调节（280px ~ 600px），展开/收起状态持久化到配置文件。
- **使用指南对话框**：内置使用指南，帮助用户快速了解功能与操作方式。
- **盘中同步确认弹窗**：A 股交易时段内同步时弹出自定义主题弹窗提示，替代原生对话框。
- **数据备份/恢复**：支持手动备份与从备份文件恢复数据库。

### Changed
- AI 接口模板重命名：`openai | azure` → `completion | responses`，旧配置自动迁移。
- 移除模型编辑中的 Temperature 输入框（深度思考模型已不需要此参数）。
- AI 配置弹窗优化：供应商列表宽度增至 240px，弹窗高度调整为窗口 90%；新增供应商后自动滚动定位；输入框支持一键清空；添加模型时名称留空；打开弹窗自动定位到当前所选模型对应的供应商；允许删除最后一个模型。
- AI 配置弹窗模型栏已移除，改为输入区模型选择器。
- 欢迎页重构：三步引导流程（选目录 → 确认主题 → 首次同步 Loading），自适应宽度，切换动画。
- 同花顺目录选择：新增目录解析功能，可识别目录类型（用户目录 / 安装根目录），提供选择提示。
- 菜单栏按钮顺序优化：AI 面板 → 字体 → 指南 → 主题，AI 按钮带活跃状态指示。
- Toast 提示颜色改为主题主色调，提升可见度。
- AI 输入框发送按钮内嵌为向上箭头图标，输入框支持 Shift+Enter 换行自动增高（最高 9 行）。
- 模型选择按钮和板块名称文字跟随字体大小设置变化。
- dev 脚本改用 `electron-vite dev` 启用 HMR 热更新，修改 renderer 代码无需重启。
- 新增 GitHub Actions release workflow，推送 tag 时自动打包并发布到 GitHub Releases。
- pnpm-workspace.yaml 补充 `packages` 字段修复 CI 构建失败。

### Fixed
- **模型条目展开后底线不可见**：`overflow: hidden` 改为 `overflow: clip` 避免裁切自身 border；滚动逻辑改用双重 rAF + 手动定位，等待展开内容渲染后再滚动。
- **AI 面板宽度丢失**：修复折叠 AI 面板后再展开时宽度恢复为默认值而非上次拖拽宽度的 bug。
- **启动同步竞态**：修复当日首次运行时同步完成后页面数据不刷新的问题（改用 renderer-ready 信号驱动同步，避免 sync-done 事件丢失）。
- **重置日期段查询**：修复重置后查询参数未正确更新的问题，同步数据后自动触发默认查询。
- **首次安装无数据**：修复首次配置完成后主界面无数据展示的问题。

## [1.0.0] - 2026-06-16

### Added
- 读取同花顺自定义动态板块配置（`stockblock.ini`，GB18030 编码）。
- 自动搜索 / 手动选择同花顺用户目录。
- 通过腾讯行情接口批量获取 A 股实时行情，失败自动回退本地缓存。
- 计算各板块的股票数量、平均涨跌幅、平均股价、平均成交额、总成交额、平均换手率。
- 基于 SQLite（sql.js）持久化历史数据，退出前自动备份。
- ECharts 可视化：柱状图（总成交额）+ 平滑曲线（其他指标），数据项可点击切换高亮。
- VSCode 风格五区布局：顶部菜单栏、左侧主侧栏、中间图表区、右侧辅侧栏、底部状态栏。
- 明亮 / 暗色双主题，默认暗色，选用无障碍配色。
- NSIS 安装程序，卸载时可选清理用户数据目录。
- 集成 electron-log 统一日志系统。
