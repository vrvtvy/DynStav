# 安全策略

## 报告安全漏洞

我们非常重视 DynStav 的安全问题。如果你发现安全漏洞，请**不要**通过公开 Issue 提交，而是私下告知我们，以便我们在公开前先修复。

请通过以下方式之一报告：

- **GitHub Security Advisory**（推荐）：前往仓库的 **Security** 标签页 → **Report a vulnerability**。
- 或发送邮件至仓库维护者。

报告时请尽量包含：

- 问题的清晰描述与潜在影响
- 复现步骤 / PoC
- 受影响的版本

我们会在收到报告后尽快确认并跟进。

## 支持的版本

由于项目处于早期阶段，我们只为**最新版本**提供安全更新。

| 版本 | 是否支持 |
|------|----------|
| 最新 release | ✅ |
| 旧版本 / 开发分支 | ❌ |
